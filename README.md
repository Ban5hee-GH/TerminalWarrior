# FileSystem-CTF
A command-line game where you explore a simulated Linux filesystem and solve hacking-themed puzzles using real Linux commands.

## Features
- Realistic Linux Commands ('ls', 'cat', 'cd', etc.)
- Fake file system to explore.
- Hidden passwords and SSH-like mini challenges.
- Educational focus on command-line and ethical hacking basics.

## Try it locally
---bash
- Step 1: (git clone https://github.com/Ban5hee-GH/FileSystem-CTF.git)
- Step 2: (cd FileSystemCTF)
- Step 3: (python cli_lab/main.py)
